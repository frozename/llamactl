# novaflow — UI kit

Creative canvas for generative brand design. Demonstrates Beacon tokens applied to a bright creative tool (Ember theme) rather than an operator console.

**Patterns demonstrated:**
- Top bar (48px): brand · crumb · tool palette (center) · primary actions
- Tool chips with shortcut glyphs and glowing active state
- Layers panel with thumbnail squares, eye toggles, indentation
- Dark canvas with dotted grid, artboard with elevation, selection frame + 8 handles
- Inspector panel: transform · typography · fill · effects — numeric fields with unit suffixes, segmented weight picker, color rows with opacity
- Timeline footer: transport controls, timecode with orange accent, track with layered clips + playhead
- Zoom chip pinned bottom-right of canvas

Tool shortcuts (V/F/T/P/R/I/M/C) are wired — press to swap tools.

Cosmetic recreation on Beacon tokens. The canvas art is hand-assembled — no real layout engine behind it.
