# llamactl — UI kit

Web control plane for LLM fleet operations. Dense, terminal-adjacent, built on Beacon's Sirius theme.

**Patterns demonstrated:**
- Title bar with traffic lights, brand orb, command-path center piece, theme-switch area
- Activity rail (left 56px) + explorer (280px) + editor surface
- Tabs with brand-accented top border
- Stat cards · 4-up with sparklines and deltas
- Data table with mono columns, status badges, tabular numerals
- Streaming logs view with severity colors
- Command palette (⌘K) overlay with pulsing carret orb
- Status bar (bottom 26px) — branch, problems, region, version

Open `index.html`, press ⌘K (or click the center bar), or switch between Services / Logs in the tabs.

This is a cosmetic recreation built on the tokens — not a real control plane. There's no real telemetry, just representative fixture data.
